  <h1>Simple C++ Calculator</h1>


A minimal command-line calculator written in C++.  
It supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
